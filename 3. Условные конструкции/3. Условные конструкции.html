Условный оператор if-elif-else

if логическое_выражение:
    операторы...
[elif логическое_выражение:
    операторы...]
[else: 
    операторы...]

Логическое выражение состоит из логических операторов и операций.

Логические операторы: > < == != >= <=

Логические операции: and or not

age = 18
if age >= 18:
    print("Вы совершеннолетний")
else:
    print("Вы несовершеннолетний")

score = 75
if score >= 90:
    print("Отлично")
elif score >= 70:
    print("Хорошо")
elif score >= 50:
    print("Удовлетворительно")
else:
    print("Неудовлетворительно")

Логические операции and и or

С помощью операций and и or можно объединять несколько логических выражений в одном условии.

  • and (логическое «И») — условие истинно, только если оба выражения истинны.
  • or (логическое «ИЛИ») — условие истинно, если хотя бы одно из выражений истинно.
if выражение_1 and выражение_2:
    ...

if выражение_1 or выражение_2:
    ...
age = 20
has_ticket = True

if age >= 18 and has_ticket:
    print("Вход разрешён")
else:
    print("Вход запрещён")

day = "суббота"

if day == "суббота" or day == "воскресенье":
    print("Выходной день")
else:
    print("Рабочий день")

Операции можно комбинировать. Для управления приоритетом используются скобки:

age = 17
has_parent = True
has_ticket = True

if (age >= 18 or has_parent) and has_ticket:
    print("Вход разрешён")
else:
    print("Вход запрещён")

Указание диапазона значений в условии

Вместо условия x > 10 and x < 20 можно использовать сокращённую запись:

if 10 < x < 20:
    print('x лежит в диапазоне от 10 до 20')

Тернарный оператор

Тернарный оператор позволяет записывать простые условные конструкции в одну строку. Это делает код более компактным и читаемым, когда нужно присвоить значение переменной в зависимости от условия.

переменная = значение_если_true if логическое_выражение else значение_если_false
a = 10
b = 3
max_num = a if a > b else b

Конструкция match

Конструкция match (введена в Python 3.10) предоставляет мощный способ сопоставления с образцом. Она позволяет сравнивать значение переменной с различными шаблонами и выполнять соответствующий блок кода, что похоже на оператор switch в других языках программирования, но с более широкими возможностями.

match выражение:
    case шаблон_1:
        операторы...
    case шаблон_2:
        операторы...
    ...
    case _:
        операторы_по_умолчанию
day = 2
match day:
    case 1:
        print('Понедельник')
    case 2:
        print('Вторник')
    case 3:
        print('Среда')
    case 4:
        print('Четверг')
    case 5:
        print('Пятница')
    case 6:
        print('Суббота')
    case 7:
        print('Воскресенье')
    case _:
        print('Неверный номер дня')

Объединение шаблонов

Можно указывать несколько шаблонов в одном блоке case, используя оператор | (или):

status = 404
match status:
    case 400 | 404 | 500:
        print("Ошибка")
    case 200:
        print("OK")
    case _:
        print("Неизвестный статус")

Распаковка последовательностей

Конструкция match позволяет не только проверять значения, но и распаковывать структуру данных (например, списки или кортежи):

point = (0, 5)
match point:
    case (0, 0):
        print("Точка в начале координат")
    case (0, y):
        print(f"Точка на оси Y: y={y}")
    case (x, 0):
        print(f"Точка на оси X: x={x}")
    case (x, y):
        print(f"Точка с координатами: ({x}, {y})")

Условия в шаблонах

К шаблону можно добавить условие if, которое будет проверяться только если шаблон совпал: 

num = 5
match num:
    case x if x % 2 == 0:
        print(f"{x} — четное число")
    case x:
        print(f"{x} — нечетное число")
