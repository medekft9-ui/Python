Расстояние между точками

Задание 101 Введение в Python
Сложность
Легкая

Описание задачи

На вход программы подаются координаты двух точек. Необходимо вычислить расстояние между ними.

Входные данные

Четыре вещественных числа: x₁, y₁, x₂, y₂ - координаты двух точек. Каждое значение с новой строки.

Выходные данные

Одно число - расстояние между точками.

Формулы

Формула для вычисления расстояния между двумя точками:

$$d = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2}$$

Подсказки

Используйте функцию sqrt() для вычисления квадратного корня

Не забудьте импортировать модуль math
